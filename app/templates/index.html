Project Introduction

Weather Web App and Exploration into AWS

Intuition

The scope of this project is to explore popular cloud platforms that the industry uses nowadays. Personally, I have picked the Amazon Web Services (AWS) for its powerful SaaS supports and its reputation in the industry. To be able deploy my app, I familiarized myself with relevent AWS services such as EC2 and RDS.


Technologies Stack of the Web App
  • Python Flask for backend web development
  • Bootstrap and D3 for frontend styling and DOM tree manipulations
  • PostgreSQL for database management
  • OpenWeatherMap for real time weather data acquisition


Cloud Technologies Involved
  • AWS EC2 for hosting the web service and the ETL service
  • AWS RDS for relational database management
  • Nginx Maybe?
